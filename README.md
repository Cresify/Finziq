# Finziq

Finziq es una aplicación móvil enfocada en el control inteligente de finanzas personales.

Permite registrar ingresos, gastos y movimientos de ahorro/inversión de forma clara, simple y rápida.

---

## Características principales

- Registro de ingresos y gastos
- Gestión de ahorro e inversión (con retiros y depósitos)
- Presupuestos mensuales por categoría
- Dashboard con:
  - Ahorro del mes
  - Capital acumulado
  - Distribución de gastos
  - Dinero disponible
- Histórico anual de capital
- Soporte multimoneda (configurable)
- Funciona offline

---

## Tecnologías

- React
- TypeScript
- Vite
- Tailwind CSS
- Capacitor (Android)
